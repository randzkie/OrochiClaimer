How This Works
1
Click "Claim Airdrop"
Tool calls the API: /airdropChallenge?address=0xYou
2
API returns ECDSA proof + payload
Signature, amount, nonce, timestamp — lahat galing sa operator.
3
Tool submits claim() on-chain
Contract verifies signature, checks 5-min expiry, transfers tokens to you.
Important
Ang proof ay valid lang sa loob ng 5 minutes. Kung hindi mo na-submit before mag-expire, kailangan mo ulit mag-fetch ng bagong proof.

Ang ECDSA proof ay nagbabago bawat fetch dahil nagbabago ang timestamp at nonce. Hindi mo pwedeng gamitin ulit ang lumang proof.
